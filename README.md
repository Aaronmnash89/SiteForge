# SiteForge

A reusable, JSON-driven website framework built with Next.js and TypeScript.

## Features

- JSON-powered content
- Modular sections
- Reusable UI components
- Multiple industry templates
- Theme support
- Responsive design
- Type-safe configuration

## Current Industries

- Personal Injury Law (Complete)

## Planned

- Roofing
- HVAC
- Plumbing
- Dental
- Landscaping
- Auto Repair

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Lucide React

## Development

npm install

npm run dev
